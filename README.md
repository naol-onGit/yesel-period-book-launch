# 📖 Yesel Period – Book Launch Website

## 📌 Overview

**Yesel Period – Book Launch** is a modern, responsive promotional website built to support and showcase the launch of the *Yesel Period* book.

The platform provides event information, book details, and structured presentation content in a clean and engaging user interface.

This project demonstrates frontend architecture, responsive UI design, and structured content management using modern web technologies.
